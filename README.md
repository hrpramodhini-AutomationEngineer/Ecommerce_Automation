# 🛒 Ecommerce Automation Framework (SDET Project)

## 📌 Project Overview

This is an end-to-end test automation framework built using **Selenium + Python + PyTest** following **Page Object Model (POM)** design pattern.

## 🚀 Features

* Page Object Model (POM) structure
* PyTest fixtures & reusable functions
* End-to-End test scenarios
* Logging & Screenshot capture on failure
* Clean and scalable framework

## 🛠️ Tech Stack

* Python
* Selenium WebDriver
* PyTest
* Git & GitHub

## 📂 Project Structure

```
pages/        -> Page classes (Login, Inventory, Cart)
tests/        -> Test cases (E2E)
utils/        -> Logger & Screenshot utilities
conftest.py   -> Fixtures
pytest.ini    -> Config
```

## ▶️ How to Run Tests

```bash
pip install -r requirements.txt
pytest -v
```

## 📸 Screenshots
